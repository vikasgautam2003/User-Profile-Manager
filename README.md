# 👤 User Manager Backend App

A simple and efficient **CRUD API** built using **Node.js**, **Express.js**, and **MongoDB** to manage user data. This backend app allows you to **create**, **read**, **update**, and **delete** user records.

---

## 🚀 Features

- 📥 Create a new user
- 📤 Read all users or a single user by ID
- ✏️ Update user details
- ❌ Delete a user
- 🧪 Basic error handling
